1. install node.js
   go to - http://nodejs.org/  

2. install: npm install -g generator-backbone

3. Make a new directory and cd into it:
   mkdir appstore && cd $_

4. Run [yo backbone], optionally passing an app name:
   yo backbone [app-name]

5. install the CLI
   npm install -g grunt-cli (sudo maybe)

6. install grunt in project folder and adding it to your devDependencies:
   npm install grunt --save-dev

7. install 'connect-livereload' module
   npm install --save-dev connect-livereload
   npm install

8. install less
   npm install grunt-contrib-less --save-dev

9. grunt serve
